# RDL support matrix

The first grouped-sales vertical slice is implemented on `spike/first-real-rdl-generation`. It generates a real RDL 2016/01 file with 24 synthetic `ENTERDATA` rows, grouped tablix, subtotals/grand totals, pagination, backup, SHA-256 evidence, well-formedness checks, Microsoft-XSD validation, deterministic structural checks, and a golden file. Report Builder Preview and PDF/Excel export remain pending independent Windows validation.
