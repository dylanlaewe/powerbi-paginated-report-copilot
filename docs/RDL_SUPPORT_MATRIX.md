# RDL support matrix

The original scratch-generated grouped report failed to open in Report Builder. The completed compatibility ladder proves template-derived reports through Candidate 06b: embedded data, detail columns, grouping, Region subtotals, Grand Total, explicit Letter dimensions, Region page breaks, repeated headings, Page N of M, Preview, PDF, and Excel all passed independent Windows validation. Scratch construction of fragile tablix hierarchies remains unproven and unsupported. `.rdl -text` preserves checksum-addressed bytes across Git platforms.
