# Product vision

Build a polished companion that understands an existing Power BI semantic model and report, turns plain-language requests into an inspectable build plan, safely modifies real PBIP/PBIR/TMDL files, validates references and layout, and provides exact diffs and restore. A constrained real RDL authoring path follows the first standard-report vertical slice. Analysis and linting support authoring; they are not the primary product.
