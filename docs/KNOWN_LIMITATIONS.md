# Known limitations

- The bootstrap UI is informational; project selection is deliberately disabled until path containment and discovery exist.
- No report or semantic-model authoring is implemented.
- No AI provider is implemented; no credential is required for the deterministic offline path.
- PBIR remains preview and no schema revision is pinned yet.
- Power BI Desktop and Report Builder are unavailable on macOS; rendering claims require independent Windows evidence.
- Candidate 03b proves a preserved Report Builder-authored grouped hierarchy, not generalized programmatic group construction. Its duplicated Region/Region1 columns and heavy wrapping are deferred presentation issues.
- Candidate 04 Region subtotals are structurally validated but not yet independently opened or previewed.
