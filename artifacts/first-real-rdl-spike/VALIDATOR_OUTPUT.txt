Compatibility note:
ReportDefinition-2016.official.xsd is the verbatim Microsoft XSD extracted from that page. Microsoft's standalone XSD does not compile in libxml because optional AuthoringMetadata references an unimported namespace type. ReportDefinition-2016.xsd changes only that unused declaration's type to xsd:string; the generated report contains no AuthoringMetadata element. No other schema rule is changed.
